# tools/

Helper scripts organized into subfolders.

## Audit (npm run audit:*)

- audit:links - game.link reachability
- audit:sitemap - sitemap URLs 200
- audit:imgs - imgSrc icon reachability
- audit:desc - description coverage

## Subfolders

- audit/ - network probes
- content/ - SEO & sitemap generation
- debug/ - Puppeteer automation
- deploy/ - purge-cloudflare, verify-build
- legacy/ - superseded Python tools
